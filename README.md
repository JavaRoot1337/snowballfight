# SnowballFight

SnowballFight - плагин для Minecraft, добавляющий эффекты к снежкам. При попадании в цель накладывается эффект замедления, воспроизводятся звуковые и визуальные эффекты. Реализована система кулдауна и настраиваемые параметры.

## Особенности

- Система кулдауна для снежков
- Эффекты замедления при попадании снежка в игрока
- Звуковые эффекты при попадании
- Визуальные эффекты партиклов при попадании
- Настраиваемые параметры через конфигурацию
- Команды перезагрузки конфигурации

## Команды и права

Команды:
- /snowballfight reload - перезагрузка конфигурации (требует права snowballfight.reload)
- /sbf reload - альтернативная команда перезагрузки (требует права snowballfight.reload)

Права:
- snowballfight.command - использовать команды плагина
- snowballfight.reload - перезагружать плагин

## Эффекты

При попадании снежка:
- Наложение эффекта замедления на цель
- Воспроизведение звукового эффекта
- Отображение визуальных эффектов (снежинки, блестки, огоньки)

## Конфигурация

Плагин настраивается через файлы config.yml и message.yml. Примеры конфигурации приведены в соответствующих файлах в папке плагина.
# Настройки эффектов снежка
snowball_effects:
  slowness:
    duration: 3  # длительность эффекта замедления в секундах
    amplifier: 4  # сила эффекта (уровень 5, так как начинается с 0)

# Настройки кулдауна снежка
snowball_cooldown:
  enabled: true # включить/выключить кулдаун
 duration: 2  # длительность кулдауна в секундах

# Настройки звука при попадании снежка
snowball_sound:
  enabled: true  # включить/выключить звук
  sound_type: "minecraft:entity.firework_rocket.blast"  # тип звука при попадании снежка

# Настройки партиклов
particle_effects:
  multiplier: 1  # множитель количества партиклов (1 = стандартное количество, 2 = в 2 раза больше и т.д.)
```

Также доступна настройка сообщений в файле `message.yml`:

```yaml
# Файл с сообщениями плагина SnowballFight
reload_success: "&aПлагин перезагружен"

command_usage: "&cИспользование: /%command% [reload]"
player_only: "&cТолько игрок может выполнить эту команду!"

## Использование

### Для игроков:
- Просто бросайте снежки как обычно — они будут работать с наложенными эффектами
- При попадании в цель будет наложен эффект замедления
- Визуальные и звуковые эффекты активируются автоматически

### Для администраторов:
- `/snowballfight reload` или `/sbf reload` — перезагрузить конфигурацию плагина
- Для использования команды необходимы права `snowballfight.reload`

## Команды

| Команда | Описание | Требуемые права |
|--------|----------|----------------|
| `/snowballfight reload` | Перезагружает конфигурацию плагина | `snowballfight.reload` |
| `/sbf reload` | Сокращенная версия команды перезагрузки | `snowballfight.reload` |

## Права

| Право | Описание |
|-------|----------|
| `snowballfight.command` | Позволяет использовать команды плагина |
| `snowballfight.reload` | Позволяет перезагружать плагин |

## Эффекты

- **Замедление** — накладывается на цель при попадании снежка
- **Партиклы** — красивые визуальные эффекты при попадании снежка:
  - Снежинки
  - Блестки разного цвета
  - Огоньки
- **Звук** — воспроизводится при попадании снежка
